Get Cleaned Data Open on Hyperskill

You have a form for using a promo code in an online shop:

class PromoCodeForm(forms.Form):
    user = forms.CharField()
    code = forms.CharField(min_length=10, max_length=10)

Write a function to validate the form and get cleaned data. If the input is not valid, you should return an empty dictionary.

The single argument of the function is raw_data. You can initialize the form directly with it.
