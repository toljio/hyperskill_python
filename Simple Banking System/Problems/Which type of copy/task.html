Which type of copy? Open on Hyperskill

You found a machine that can copy objects. It is a device that takes an object and returns its copy. The problem is that you don't know if it creates a shallow copy or a deep copy. You should solve this problem.

You should write a function detect_copy() which takes no arguments. From this function, you can call the predefined function copying_machine(obj) several times; it takes the passed object obj and returns either its shallow copy or its deep copy. The copying machine function makes only one type of copy, and your function should find this out and return "shallow copy" or "deep copy".

You do NOT need to read anything from the input, define obj for yourself. You can call copying_machine() in the following way: copy_obj = copying_machine(obj).
