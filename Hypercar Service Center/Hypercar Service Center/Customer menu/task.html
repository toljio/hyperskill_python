Description

In the Hypercar service center customers can always find help with three problems:

  • Change oil
  • Inflate tires
  • Get diagnostic test

To be able to make a choice, a customer should have a menu. Each paragraph in the menu is the descriptions of the service. After clicking on an item the client is redirected to the proper page for getting a ticket, so use special HTML elements to prepare these links.

Objectives

Now your task is to make a menu page with three <a> elements with the href attribute equal to "/get_ticket/change_oil", "/get_ticket/inflate_tires", "/get_ticket/diagnostic" accordingly:

<a target="_blank" href="/get_ticket/inflate_tires">Inflate tires</a>

Take the description for each link from the list above.

The menu page should be available on "<base_url>/menu", where <base_url> is the host. To combine a template with the HTTP handler you can use django.shortcuts.render function. To pass any arguments to template add the dictionary to the render:

render(request, template_name, {'name_you_use_in_template': your_variable})

You should not implement handlers for these pages: you'll do it at the next step.
